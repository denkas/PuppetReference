Puppet, MCollective, Facter, and Dashboard.

  • Puppet is open source software for configuring and managing your infrastructure.
  • MCollective is a framework to build server orchestration or parallel job execution systems.
  • Facter is an independent, cross-platform Ruby library designed to gather information on all the nodes you will be managing with Puppet.
  • The Puppet Dashboard is a web interface and reporting tool for your Puppet installation.
